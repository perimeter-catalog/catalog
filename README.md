# Automated Perimeter Archive

A reference archive of commercial automated perimeters (visual field testing instruments),
cataloging historical and contemporary devices with detailed technical specifications,
test pattern figures, linked literature, and supporting assets.

Live site: https://perimeter-catalog.github.io/catalog/

## About

The archive aims to preserve detailed product knowledge of automated perimetry devices
(1972–present) for research and clinical reference. Each device record contains:

- Manufacturer, release year, and country of origin
- Test patterns (with extracted geometry as Pattern Figure records)
- Test strategies and analysis capabilities
- Reliability framework references
- Primary source citations (peer-reviewed literature, manufacturer documentation)

Family Overview records aggregate series-level information for related model lineages
(e.g. Humphrey Field Analyzer I, II/II-i, 3).

## Data source

Records live in an Airtable base; this site is a static client-side renderer.
